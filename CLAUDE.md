# ミニマムタックス M&Aシミュレーター

## プロジェクト概要

2026年度（令和8年度）税制改正によるミニマムタックス強化が、M&Aにおけるオーナー社長の株式譲渡後の手取り額に与える影響を可視化するWebアプリ。

M&Aアドバイザーがクライアント（オーナー社長）に対し、売却タイミングや手取り額への影響を説明するために使用する。

## 技術スタック

- **フレームワーク**: React 18 + Vite
- **言語**: TypeScript（推奨）/ JavaScript
- **チャート**: Recharts
- **スタイリング**: Tailwind CSS
- **PDF出力**: 将来対応（html2canvas + jsPDF）
- **デプロイ**: 静的ホスティング（Vercel / Netlify）

## 計算ロジックの核心（taxEngine.ts）

### ミニマムタックスの計算式

**改正前（〜2026年分）**:
```
追加税額 = max(0, (基準所得金額 − 3億3,000万円) × 22.5% − 基準所得税額)
```

**改正後（2027年分〜）**:
```
追加税額 = max(0, (基準所得金額 − 1億6,500万円) × 30% − 基準所得税額)
```

### 重要な計算上の注意点

1. **基準所得金額**: 所得控除「前」の全所得合計（総合課税所得 + 分離課税所得）
2. **基準所得税額**: 所得控除「後」の税額合計（累進税率の総合課税分 + 15%の分離課税分）
3. **住民税にはミニマムタックスは適用されない**（所得税のみ）
4. **復興特別所得税**: (通常所得税 + MT追加税額) × 2.1%
5. **所得控除**: 総合課税所得から優先適用し、引ききれない分を分離課税所得から控除

### 検証済みのテストケース

| ケース | 条件 | 改正前手取り | 改正後手取り | 増税額 |
|--------|------|-------------|-------------|--------|
| TC1 | 株式5億, 取得0 | 3億9,883万 | 3億7,249万 | 2,634万 |
| TC2 | 株式10億, 取得0 | 7億9,618万 | 6億9,434万 | 1億184万 |
| TC3 | 株式3億, 取得0 | 2億3,946万 | 2億3,946万 | 影響なし |
| TC4 | 給与3千万+株式10億 | 7億9,685万 | 6億9,625万 | 1億60万 |

テストケースの詳細と計算過程は `docs/verification_output.txt` を参照。

## デザイン方針

- **トーン**: ダークネイビー基調、ゴールドアクセント、プロフェッショナル
- **対象**: クライアント（オーナー社長）への提示用
- **金額表示**: 1億円以上は「○億○,○○○万円」形式
- **会社ロゴ**: ヘッダー左上に配置（SVG差替え可能な設計）

## 注意事項

- 計算ロジック（taxEngine.ts）は検証済み。変更時は必ずテストケースで再検証すること
- 全ての計算はクライアントサイドで完結（サーバー送信なし）
- 本シミュレーションは令和8年度税制改正大綱に基づく概算であり、法案成立後に最終確認が必要
